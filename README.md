## 🧠 Descripción

Este proyecto representa la evolución completa de un sistema ecommerce, integrando frontend, backend, base de datos y seguridad en una aplicación funcional desarrollada con Spring Boot.

💡 Incluye lógica de carrito, gestión de pedidos e historial de compras, acercándose a un sistema ecommerce real.
