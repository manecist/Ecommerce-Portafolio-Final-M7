Instrucciones de ejecución

1. Descargar desde GitHub link -> https://github.com/manecist/Ecommerce-Portafolio-Final-M7
2. Extraer Zip
3. Abrir desde el IDE IntelliJ o cualquier otro IDE como Netbeans
4. Open folder -> Confió en el archivo
5. Antes de ejecutar se debe crear la base de datos llamada -> crud_ecommerce_m6;
6. Escribir en application properties la base de datos
7. Ejecutar el scrip M7 que esta dentro de la carpeta para aplicar mi lógica de tablas dentro de la base de datos
8. Voy a C:\demoCrudEcommerceM7\src\main\java\com\magicalAliance\DemoCrudEcommerceM6Application.java
9. Aplico Run desde app  o bien desde la carpeta de demoCrudEcommerceM7 en la opción terminal uso comando mvn spring-boot:run
10. Escribo en Url -> http://localhost:8080/ o http://localhost:8080/home
11. Se vera la app gracias al tomcat interno y se puede navegar en ella mientras la terminal este ejecutada.
